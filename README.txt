Este trabalho simula um porto que recebe operação de clientes e as processa para empresas, passando por intermediários.
Para compilar o programa segue-se as seguintes instuções:

Dentro da pasta ADMPOR execute os seguintes comandos por ordem:

1º make clean(vai limpar os ficheiros .o, executavel e de log e estatisticas)
2º make (onde vai gerar os ficheiros .o e executavel)
3º cd bin
3º ./admpor input.txt (substituir os argumentos pelos numeros desejados que no caso é o ficheiro com os inputs).

O projeto demonstrou um erro estranho poucas vezes, em que o programa não acaba a execução. Mas quando executado o comando make clean,
e make de novo o comportamento voltava ao normal.
O projeto foi testado e realizado utilizando o sistema WSL com Ubunto, no IDE VSCode.
